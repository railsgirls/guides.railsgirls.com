Guías Rails Girls

Estas guías estan diseñadas para brindar las herramientas y una comunidad, para que las mujeres entiendan las tecnologías para desarrollar sus ideas. Organiza tus propios eventos, puedes presentar nuevas guías o simplemente utilizalo para aprender Rails. Para más información, subscribete a la lista de correo.
